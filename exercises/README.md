These are 2 exercises to get familiar with the alpaka syntax.

For each exercise you have 2 folders:
- cuda: contains the solution writte in cuda
    - Compile running `make cuda`
- alpaka: contains a cpp file with the skeleton of the code. You should complete the exercise using alpaka. Compare the code with the CUDA one!
    - Make sure you compile for both CPU and GPU! The precompiler directive magic is in `config.h` if you want to have a look! 
        - Compile running : `make cuda serial`
